Use Guide

中文 | English

  1. Hit "Set List" to set candidates, you can import candidates from a txt file, seperate multiple names with comma or new line. Duplicated names would be auto-removed.

  2. The number of total candidates would show on the button after list set.

  3. Then please set the number of winners for this round.

  4. Hit "Start", winner names would appear on screen and change quickly. Hit "Stop", name would stop changing and these names are winners.

  5. You can use the Space key on keyboard to Start/Stop.

  6. Use "Exp & Rem" (Export & Remove) to export a txt file names under current time, which contains winners for this round and non-winner names. And winners would be removed to prevent them from being winner again for next round.
