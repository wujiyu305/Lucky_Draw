Use Guide

中文 | English

  1. Use "Input List" to input candidate names, seperate miltiple names with comma. If got a big list, it was suggested to use "Import List", it could import candidates from a txt file, seperate multiple names with comma or new line. Duplicated names would be auto-removed.

  2. The number of total names would show after list set. Then please input the number of winners this this round.

  3. Hit "Start", winner names would appear on screen and change quickly. Hit "Stop", name would stop changing and these names are winners.

  4. You can use the Space key on keyboard to Start/Stop.

  5. Use "Exp & Rem" (Export & Remove) to export a txt file names under current time, which contains winners for this round and non-winner names. And winners would be removed to prevent them from being winner again for next round.
